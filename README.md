# Minicosmetics


MiniCosmetics is a simple, responsive cosmetic brand landing page built using **HTML** and **CSS**.  
It focuses on a clean design, soft gradients, and an elegant look suitable for a beauty or cosmetic brand.

##  Features

- Attractive gradient-based UI
- Simple and clean layout
- Responsive hero section
- About section describing the brand
- Contact form for user messages
- Beginner-friendly code structure

## Technologies Used

- HTML5
- CSS3
